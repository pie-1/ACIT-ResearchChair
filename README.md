# Research chair members Work

This repository is used to track the daily learning and research progress of all Research Chair members.

## Purpose
# ACIT-ResearchChair

The goal of this repository is to:

- Maintain a record of daily learning.
- Share knowledge with other team members.
- Track individual progress over time.
- Build a collaborative learning environment.

## Repository Structure

Each member has their own folder.

Example:

```
clubwork/
├── kabin/
├── anzel/
├── raymon/
└── README.md
```

Members should only add or update files inside their own folder.

## Daily Update Guidelines

- Create a new file for each day or topic.
- Use meaningful file names.

Examples:

```
2026-07-03.md
day-01.md
git-basics.md
python-functions.md
```

Each daily note should include:

- Date
- Topic learned
- Summary of what you learned
- Resources used (if any)
- Questions or challenges (optional)

## Workflow

1. Pull the latest changes.
2. Add your daily learning file to your folder.
3. Commit your changes with a meaningful commit message.
4. Push your changes.

Example:

```bash
git pull origin main

git add .

git commit -m "Add learning for 2026-07-03"

git push origin main
```

## Rules

- Only modify files inside your own folder.
- Do not edit another member's work without permission.
- Use clear and descriptive commit messages.
- Keep updates consistent and organized.
